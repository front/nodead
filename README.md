# Instructions

#### 1 Install Node

http://nodejs.org/download/


#### Install Redis

With [Homebrew](http://mxcl.github.com/homebrew/):

    brew install redis

By hand:

http://redis.io/download

#### Clone the repository

    git clone git@github.com:front/nodead.git

#### Run the app

    cd nodead
    node app

#### Check it out

http://localhost:3000/
